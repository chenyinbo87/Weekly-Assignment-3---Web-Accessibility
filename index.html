Accessibility features

The following is a list of basic accessibility features of this web application.

  • Voiceover-provides audible output,combined with suppord for keyboard navigation
  • Keyboard shortcuts- provides keyboard shortcuts for tabs and buttons, instead using pointing devices
  • ARIA landmark- provides content roles for identifying page structure
  • ARIA live regions- provides feedback when the web page is updated
  • Image caption- provides the image describtion of the web page

Keyboard shortcuts

Common Windows Browsers: Shift + Alt + [the accesskey]
Common Mac Browsers: control + option + [the accesskey]
